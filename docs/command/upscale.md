# /upscale - 画像の高品質な拡大

## 説明

画像を高品質に拡大する　AI画像以外にも

## オプション

---

### image / 拡大する画像

---

拡大する画像（必須）

- 縦横いずれも1280pxまで
  
  - 長辺が640px以下の場合は4倍、それ以外は2倍に拡大

---

### mode / 拡大モード

---

拡大の品質

- Defaultはちょっと彩度高めになって、細部がわずかに変化する

  - 変化するといっても、良くなる場合もある
  
- Ultraはそれらの変化を限りなく少なくし、そのまま拡大する

  - それでもちょっと彩度が高くなるかも

  - 所要時間が非常に長いので、気をつけて
  
- 推定所要時間は512x512の場合のため、サイズによっては前後する
